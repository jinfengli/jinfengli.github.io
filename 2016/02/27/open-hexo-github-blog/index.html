open Hexo github blog

发表于 2016-02-27 09:45:39 | 分类于 blog |

配置github 博客遇到的问题：

Q1: 执行命令hexo server 启动后，在浏览器中通过 http://localhost:4000 无法访问？
A1: 在任务管理器中查看4000端口被哪个软件占用了，杀掉相应进程即可。 或者通过 hexo server --port=xxxx 来设置新的端口号为xxxx 。

Q2: 如何更新themes?
A2: 进入到 themes/next目录下,执行git pull ,就可以更新相应的themes

why open github blog?

  • 想提高自己的归纳总结以及表达能力；
    虽然自己在Wiz笔记上面经常对自己工作和生活进行总结（坚持记了2年多），但是总感觉能把自己的体会和总结写成博文让大家进行批评和指正更需要勇气，更能督促自己用心去写，提升自己表达能力；

  • 结交一些有共同爱好的朋友；
    想通过写Blog梳理自己，使技术上有提升，并多认识一些小伙伴；

  • 记录自己工作、生活的点滴；

  • 折腾；
